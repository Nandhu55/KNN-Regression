# KNN Regression Prediction App

A simple Streamlit web application for predicting values using a trained KNN Regression model.

---
live app :
https://knn-regression.streamlit.app/


# Project Overview

This project uses:

- KNN Regression Algorithm
- Streamlit for Web Interface
- Pickle for Model Serialization
- Pandas and NumPy for Data Processing

The model predicts output values based on user inputs such as:

- Age
- Income
- Loan Amount
- Credit Score
- City
- Employment Type

---

# Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
